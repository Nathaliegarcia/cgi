# OmniTools

Pre-built distribution of [omni-tools](https://github.com/iib0011/omni-tools).

## Version

**Current Version: 0.6.0**

## Build Configuration

- `base: './'` - relative asset paths
- `basename="/omnitools"` - React Router configured for /omnitools/ subfolder
- Relative locale paths
- Single CSS + app/vendor JS bundles
- Only English (en) and French (fr) locales
